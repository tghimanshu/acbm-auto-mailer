# ACBM Auto Mailer

An Auto Emailer for ACBM

## Description

An Auto Emailer for ACBM created to email certificates.

## Getting Started

### Dependencies

* python 3.9+

### Installing

```
pip install pandas
```

```
pip install smtplib
```

```
pip install email
```

### Executing program

```
python auto-mailer.py
```
